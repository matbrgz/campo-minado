# Campo Minado - C CLI

Implementação clássica do jogo Campo Minado em C puro para linha de comando.

## Descrição

Esta é uma versão simples e direta do Campo Minado desenvolvida em C, executada diretamente no terminal. O jogo oferece uma experiência nostálgica e leve, perfeita para quem aprecia jogos de terminal.

## Requisitos

- GCC (GNU Compiler Collection)
- Sistema operacional Linux/Unix ou WSL

## Como Usar

### Compilar

```bash
make build
```

### Executar

```bash
make run
```

### Limpar arquivos compilados

```bash
make clean
```

### Ver ajuda

```bash
make help
```

## Funcionalidades

- Interface de texto no terminal
- Jogo completo de campo minado
- Compilação simples e rápida
